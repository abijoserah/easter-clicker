## 🎯 Objectif du challenge

Créer en 24h un petit jeu fun en JavaScript sur le thème de Pâques 🐰  
Le principe : cliquer le plus vite possible sur des éléments qui apparaissent aléatoirement à l’écran pour faire le meilleur score.

⚡ Pas de pression : l’objectif est de **tester, s’amuser et expérimenter** !

## 🕹️ Fonctionnalités principales

### 1. Niveaux de difficulté

Le jeu propose 3 niveaux :

- 🐥 **Petit poussin** (facile)
- 🐰 **Lapin malin** (intermédiaire)
- 🥚 **Chasseur d’œufs** (difficile)

Chaque niveau peut jouer sur :

- Le nombre d’objets affichés
- Le temps de jeu
- La vitesse d’apparition

---

### 2. Objets à cliquer

- 🍫 **Œufs en chocolat** → +1 point  
- ✨ **Œufs dorés** → +5 points (bonus)  
- 🐔 **Poule / œuf cassé** → -2 points (malus 😈)

Les objets apparaissent aléatoirement et disparaissent rapidement.

---

### 3. Score

- Score affiché en direct  
- Objectif : faire mieux que les autres 😏  

---

### 4. Interface

- Sélection du niveau  
- Timer visible  
- Score visible
